# Gym-desk Project: Super Admin Features

## Overview
Gym-desk is a comprehensive gym management system. The Super Admin panel is designed for platform owners to manage multiple gyms (tenants), track revenue, and monitor subscription health.

## Core Features
- **Centralized Dashboard**: Real-time stats on Gym counts, Financials (Daily/Monthly/Yearly revenue), and Subscription statuses.
- **Dynamic Analytics**: Interactive charts showing revenue trends with custom time filters.
- **Multi-tenant Management**: Full control over gym registration, branding (logos), and administrator accounts.
- **Subscription Lifecycle Tracking**: Automated tracking of active, expiring, and expired subscriptions.
- **Payment Orchestration**: Manual and automated recording of gym payments/subscriptions.

## Technical Skills Demonstrated
- Laravel 11/Eloquent ORM
- Multi-tenancy Architecture
- Livewire for dynamic UI components
- RESTful API design for data visualization
- Secure Authentication & Role-Based Access Control (RBAC)
